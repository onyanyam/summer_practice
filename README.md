## Настройка авторизации
1. Установить расширение [EditThisCookie](https://chromewebstore.google.com/detail/ojfebgpkimhlhcblbalbfjblapadhbol?utm_source=item-share-cb) в Chrome.
2. Зайти на [Rutube.ru](https://rutube.ru) и войти в аккаунт.
3. Открыть расширение и нажать кнопку **"Экспорт"** (куки будут скопированы в буфер обмена).
4. Создать в проекте в папке `src/test/resources/` файл с именем `cookies.json`.
5. Вставить содержимое буфера обмена в этот файл.

## Запуск тестов:
### Все тесты
```bash
mvn clean test
```
### Конкретный класс тестов
```bash
mvn test -Dtest=<Название_класса_тестов>
```
### Конкретный тест
```bash
mvn test -Dtest=<Название_класса_тестов>#<Название_теста_внутри_класса>
```
#### Примеры:
```bash
mvn test -Dtest=SearchTests
mvn test -Dtest=SearchTests#applyFilters
mvn test -Dtest=SearchTests#changeSearchQuery
```
